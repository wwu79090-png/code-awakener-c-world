# Contributing

Thanks for helping improve Code Awakener.

## Project Rules

- Keep the first world strictly focused on C language learning.
- Add tests before implementation for new runtime behavior.
- Run `node programming-rpg-c-basics.test.js` and `node build-single-html.cjs` before opening a pull request.
- Put course content in data/config structures rather than hardcoding it deep in scene logic.

## Good Contributions

- New C lesson variants.
- Accessibility fixes.
- Performance improvements with measured evidence.
- SDK examples for teachers.
- Documentation for classroom use.

## Pull Request Checklist

- The change has a clear learning purpose.
- Save data remains backward compatible.
- The browser version still runs without desktop APIs.
- New user input is escaped before rendering.
